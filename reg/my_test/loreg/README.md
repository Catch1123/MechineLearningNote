# 简单逻辑回归

### 逻辑回归本质上可以理解为多体费米子模型的——费米面拟合问题。

从量子统计物理出发，我们知道，费米子满足泡利不相容原理（每个状态仅能被一个全同粒子占据）。对于希尔伯特空间维度为2的费米子，比如多体电子系统，每个能级的占据情况仅存在两种可能，即填充（1）/空穴（0），即占据数为1/0。这事实上就对应了逻辑回归中的labels-`True`/`False`。

基于上述setup，我们假设系统一共存在$d$个能级，一共$G = \sum_{i=1}^d g_i $个状态（$g_i$代表第$i$个能级的简并度，也就是有$g_i$个状态满足正交化条件，但是具有相同的能量$\epsilon_i$）。现在让我们在我们的系统中引入总数为$N=\sum_i n_i$的电子数（训练样本的个数，其中$n_i$代表第$i$个能级的粒子数）。现在追问一个问题：在热力学温度为$\beta=1/(k_BT)$，化学势为$\mu$的情况下，系统每个能级（伴随着不同的能量）平均而言能被多少个粒子占据？

直觉上讲，能量越高越不稳定。同时考虑到泡利原理，电子总会优先占据低能级，然后像砌砖头一样，一层一层往高能级摞。当每个电子都有一个坑位的时候，我们直觉上讲这就是系统总能量最低的时候。而此时的总能量被称为费米能（$\beta\rightarrow\infty$时）。事实上，这个能量跟逻辑回归中的决策边界是一回事。

那么让我们详细推导一下，不同能级被占据的概率分布。既然是能级的概率分布，我们的自变量自然是能级的能量$\epsilon_i$，我们的目标是推出一个能量到占据概率的映射$p(\epsilon_i)$。

从最大熵原理出发，我们认为，电子在这种情况下的分布必然是使得熵最大的分布。因此，我们要从热力学对熵的定义出发。假设我们有一个分布$\{n_i\}$，那么熵应该是这个分布的离散泛函：

$$S[\{n_i\}] \propto \log (\Omega_\text{F.D})$$

$\Omega_\text{F.D}$为微观状态数。它的意义也很直观：熵就是系统混乱程度的度量。系统的微观状态数目越多，自然熵就越大。就好比，一把整齐排列的筷子，此时熵是最小的。随着第一拨动的加入，所有筷子随机倒向四面八方，微观状态数，此时也就是筷子头指向的方向，自然是很多的。

那么，让我们计算一下微观状态数，即给定分布$\{n_i\}$，它所有可能的微观状态。事实上，就是给$g_i$个箱子，$n_i$个完全一样的球，每个箱子放一个球，共有多少种放法的问题，就是组合数$C_{g_i}^{n_i}$（假设箱子的数目比球多得多）。对所有的能级，我们把每个能级的状态数相乘得到总的微观状态数：

$$\Omega = \prod_i^d \frac{g_i!}{n_i!(g_i-n_i)!}$$

而后，带入熵：

$$\begin{aligned}
S[\{n_i\}] &\propto \log \left( \prod_i^d \frac{g_i!}{n_i!(g_i-n_i)!} \right) \\
&= \sum_i^d \left[ \log g_i! - \log n_i! - \log (g_i-n_i)! \right] \\
&\approx \sum_i^d \left[ g_i\log g_i - n_i\log n_i - (g_i-n_i)\log (g_i-n_i) \right]
\end{aligned}$$

> 这里用到了斯特林公式：$\log n! \approx n\log (n-1)$

继续，让我们此时考虑一些约束：

- 能量守恒： $E = \sum_i n_i \epsilon_i$
- 粒子数守恒：$N = \sum_i n_i$

在这些约束下，我们对熵进行变分求极大值（拉格朗日乘子法）：

$$\begin{aligned}
\delta S[\{n_i\}] &\approx \delta \left[ \sum_i^d \left( g_i\log g_i - n_i\log n_i - (g_i-n_i)\log (g_i-n_i) \right) \right. \\
&\left. \quad + \alpha \left( \sum_i n_i - N \right) + \beta \left( \sum_i n_i \epsilon_i - E \right) \right]
\end{aligned}$$

对常数变分为0，整理得到：

$$\delta S[\{n_i\}] = \sum_i^d \left[ -\log n_i + \log (g_i-n_i) + \alpha + \beta \epsilon_i \right] \delta n_i$$

极值条件为 $\delta S[\{n_i\}] = 0$，那么自然得到：

$$-\log n_i + \log (g_i-n_i) + \alpha + \beta \epsilon_i = 0$$

整理得到：

$$\log \frac{g_i-n_i}{n_i} = -\alpha - \beta \epsilon_i$$

即：

$$n_i = \frac{g_i}{1 + e^{\alpha + \beta \epsilon_i}}$$

令 $\mu = -\alpha$，则：

$$n_i = \frac{g_i}{1 + e^{\beta \epsilon_i - \mu}}$$

写成概率形式（占据数密度 $p_i = n_i/g_i$）：

$$p(\epsilon_i) = \frac{1}{1 + e^{\beta (\epsilon_i - \mu)}}$$

写成显而易见的函数形式即：

$$\boxed{p(\epsilon_i) = \frac{1}{1 + e^{f(\epsilon_i)}}}$$

其中 $f(\epsilon_i) = \beta(\epsilon_i - \mu)$ 对应逻辑回归中的线性函数 $f(x) = w^T x + b$。当然也可以是任何复杂的函数。而 $p(\epsilon_i)$ 就是所谓的**逻辑回归中的预测概率**，即给定输入特征 $\epsilon_i$（或 $x_i$）的条件下，样本属于类别 1 的概率：

$$\boxed{p(\epsilon_i) = \frac{1}{1 + e^{f(\epsilon_i)}} = P(y_i = 1 \mid \epsilon_i)}$$

在机器学习中，这个函数被称为 **Sigmoid 函数**（或 Logistic 函数），它将线性函数 $f(\epsilon_i)$ 的输出映射到 $(0, 1)$ 区间，赋予概率解释。

更一般地，对于原始特征 $x$，若 $f(x)$ 取线性形式 $w^T x + b$，则逻辑回归的标准形式为：

$$P(y=1 \mid x) = \frac{1}{1 + e^{-(w^T x + b)}}$$

这正好对应于费米-狄拉克分布在零温极限下的版本。

### 交叉熵

## 损失函数的设计：从熵的角度

衡量训练集和模型预测的偏离程度，有很多种方法。但是作为物理学的自然延伸，我们想到：严格零温下的费米分布只有两个状态，即 $0/1$，此时的热力学熵是最小的。我们想要我们的预测模型尽可能准确地给出结果，也就是尽可能贴合精确的零温极限下的费米分布，即熵最小的状态。

由此我们可以设计一个基于熵的损失函数。回顾费米熵的表达式：

$$S_{\text{Fermi}} = -\sum_i \left[ p_i \log p_i + (1-p_i) \log (1-p_i) \right]$$

但在监督学习中，我们已知真实标签 $y_i \in \{0,1\}$，它代表零温极限下的理想占据数。因此，我们应当用 $y_i$ 代替 $p_i$ 作为"目标分布"，来衡量模型预测 $p_i$ 与目标之间的偏离程度。

受此启发，我们可以设计一个熵函数：

$$S = \sum_i \left[ (1-y_i) \log(1+p_i) + y_i \log(2-p_i) \right]$$

其中 $p_i = p(\epsilon_i) = 1/(1+e^{f(\epsilon_i)})$。

### 物理含义

- 当 $y_i = 0$（空态）时，损失项为 $\log(1+p_i)$。若模型预测 $p_i \to 0$，则损失 $\to \log 1 = 0$；若预测 $p_i \to 1$，则损失 $\to \log 2$（有限惩罚）。
- 当 $y_i = 1$（占据态）时，损失项为 $\log(2-p_i)$。若模型预测 $p_i \to 1$，则损失 $\to \log 1 = 0$；若预测 $p_i \to 0$，则损失 $\to \log 2$（有限惩罚）。

### 与标准交叉熵的对比

标准交叉熵损失函数为：

$$L_{\text{CE}} = -\sum_i \left[ y_i \log p_i + (1-y_i) \log(1-p_i) \right]$$

其特点是在 $p_i \to 0$ 或 $p_i \to 1$ 时，损失项趋向于无穷大（$\log 0 \to -\infty$），因此对错分样本施加无限大的惩罚。

而我们设计的基于熵的损失函数具有以下优点：

1. **有界性**：每个样本的损失值被限制在 $[0, \log 2]$ 之间，天然具有数值稳定性。
2. **无发散**：避免了 $\log 0$ 的数值问题，不需要额外的数值裁剪（clipping）。
3. **鲁棒性**：对异常值或噪声标签的惩罚是有限的，模型不会被少数极端样本带偏。

因此，这个损失函数在物理上对应于"有限温度下的熵最小化"，在数值上则提供了一个稳定且鲁棒的学习目标。


### 软损失函数的梯度

对 $p_i$ 求导，得到单样本损失 $L_i$ 的梯度：

$$\frac{\partial L_i}{\partial p_i} = \frac{1-y_i}{1+p_i} - \frac{y_i}{2-p_i}$$

由于 $p_i \in (0,1)$，该梯度始终有界：

$$\left| \frac{\partial L_i}{\partial p_i} \right| \le 1$$

这保证了训练过程的数值稳定性，避免了梯度爆炸问题。
